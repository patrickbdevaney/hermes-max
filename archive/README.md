# archive/ — provenance, not product

Material kept for the record. **None of it is needed to install or run
hermes-max** — see the [README](../README.md) and [QUICKSTART](../QUICKSTART.md).

- **`specs/`** — the `CLAUDE_*.md` build specifications that produced the system,
  in build order. See [specs/README.md](specs/README.md).
- **`traces/`** — generated validation traces and eval reports captured during
  development (dry-run, rate-limit, bottleneck, deep-research, emergent-eval).
  These are point-in-time artifacts; regenerate fresh ones with `hm eval`,
  `hm bottleneck`, `hm smoke`, etc.
